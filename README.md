# Base16
A metroidvania prototype made in ActionScript3 in 2012

# Controls
## Keyboard
W - jump

A - move left

D - move right

E - cycle next weapon

Q - cycle previous weapon

R - reload weapon

## Mouse
left click/hold - fire weapon

move mouse - aim weapon
